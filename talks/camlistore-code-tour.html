BlobStorage

  • put, get, enumerate
  • Lots of implementations

Schema objects

  • Simple JSON blobs with a common structure
  • All schema blobs have a camliType field
  • There are several schemas for different purposes, forming a little cake inside the big cake
    • Immutable bytes and file schemas
    • Signed permanode and claim blobs are the primatives from which mutability is built up

Index

  • Built on another low level storage interface, Sorted
  • Extracts special purpose values from blobs for quick, indexed access.
  • Totally bespoke

Clients

  • Simple command line tools: camput, camget, and a catch-all camtool
  • WebUI
  • FUSE Filesystem
  • Android and iOS apps
